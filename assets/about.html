Our Alliance

Developed by: Michael Chinn

This is a scouting app for teams to use in the FIRST Robotics Competition, it will be able to store scouting info on each team and record information about each team’s performance throughout matches in the competition. The plan is to allow multiple android devices to sync with each other.

No syncing right now but functionally it should be able to serve as a scouting app if you are a team going to an off season event!
